# Contributing to GPF-AI-OS

Thank you for your interest in GPF-AI-OS.

We welcome contributions from researchers, developers, data scientists, and AI enthusiasts around the world.

---

## Ways to Contribute

You can contribute by:

- Improving documentation
- Fixing bugs
- Adding public data connectors
- Improving reasoning modules
- Writing tests
- Optimizing performance
- Improving examples

---

## Before Contributing

Please:

- Read the project documentation
- Open an Issue before large changes
- Keep Pull Requests focused
- Write clear commit messages

---

## Coding Principles

- Keep code modular
- Prefer readability over cleverness
- Write comments where appropriate
- Avoid unnecessary dependencies

---

## Public Data First

GPF-AI-OS prioritizes public and reproducible data.

Please avoid submitting:

- Proprietary datasets
- Personal account credentials
- Brokerage account information
- Private API keys

---

## Thank You

Every contribution helps build an open AI-native research platform.
