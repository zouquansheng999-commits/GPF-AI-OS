# GPF AI OS

> AI-Native Research Operating System for Capital Intelligence

## Overview

GPF AI OS is an open-source AI-native research operating system designed to transform fragmented information into structured, evidence-based research workflows.

Rather than replacing human judgment, GPF AI OS augments professional research through AI-assisted reasoning while keeping humans responsible for all final decisions.

---

## Vision

Build an open-source research operating system powered by advanced AI reasoning models.
