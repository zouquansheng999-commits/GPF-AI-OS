# GPF-AI-OS

> AI-native Research Operating System for Capital Intelligence

GPF-AI-OS is an open-source research operating system designed to help researchers, developers, and investors understand long-term productivity growth, technological diffusion, and global capital migration using transparent public data and explainable AI.

Unlike traditional trading systems, GPF-AI-OS focuses on research, reasoning, and knowledge accumulation rather than short-term market prediction.

---

# Vision

Build an AI-native research platform capable of continuously discovering:

- Global productivity trends
- AI infrastructure evolution
- Capital allocation
- Industrial transformation
- Technology diffusion
- Long-term investment opportunities

Our goal is to create an open research infrastructure that anyone can inspect, reproduce, and improve.

---

# Design Philosophy

GPF-AI-OS is **not** intended to be a black-box trading system.

Instead, it emphasizes:

- Explainable reasoning
- Long-term research
- Knowledge accumulation
- Transparent methodology
- Reproducible analysis

Every research conclusion should be traceable back to public evidence.

# Privacy

The open-source version of GPF-AI-OS is designed around public information.

Personal brokerage accounts, API credentials, and private financial information are intentionally kept outside the public repository.

Brokerage integrations, if developed in the future, will be provided as optional private adapters rather than core project components.
