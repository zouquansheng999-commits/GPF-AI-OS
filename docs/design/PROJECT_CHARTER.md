# GPF-AI-OS Project Charter

## Mission

Build an open-source AI-native Research Operating System that discovers productivity-driven capital migration using public data and transparent reasoning.

## Vision

Become an open-source research infrastructure for understanding global productivity transformation and capital migration.

## Core Principles

- Public Data First
- Research First
- Transparent Reasoning
- Explainability
- Open Source First
- Modular Architecture
- Long-term Knowledge Accumulation

## Scope (V1)

- Public Data
- Knowledge Base
- Reasoning Engine
- Daily Radar
- Research Reports

## Non-Goals

- Brokerage Integration
- Auto Trading
- Personal Accounts
- Private Data Sources
